Analysis of DexaFit Client Booking Data

by John Akwei, Data Science Specialist

Tuesday, September 15, 2015

Synopsis

The objective of this document is analysis of 5 DexaFit booking data spreadsheets. The spreadsheet data is analyzed in order to gain an understanding of;
1) the target demographic
2) seasonal visiting trends
3) rate of growth month over month
(Chicago, San Francisco, and Atlanta only)

The data contains client identification information, and appointment-related questions, that are answered via multiple selection, and by comment from the client. The Categories of the Booking Data include Profile, Reference, Start time, Cancelled, Status, NAME, Last Name, Gender, Email, Phone Number, What are we testing this visit?, Is this a new test or a follow-up assessment?, Reason for visit, How did you learn about DexaFit?.

R programming language processing of the Data was performed to extract the conclusions in this Document. In order to increase comprehensibility of the Document, most of the R coding for data analysis was not printed as part of the Document.

The sections of this Document include:
1) Data import
2) Data Overview
3) Target Demographic Analysis
4) Seasonal Visiting Trends
5) Monthly Rate of Growth Analysis
